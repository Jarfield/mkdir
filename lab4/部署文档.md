# 单机部署
- 前提：已经安装好了python3和pip
- 安装ray
```shell
pip install -U "ray[default]"
```
- 启动ray

可以在python交互界面中启动ray
```python
import ray
ray.init()
···
ray.shutdown()
```
也可以在命令行中启动ray cluster head
```shell
ray start --head --port=6379
```
- 查看dashboard

完成启动后，可以打开网页：http://127.0.0.1:8265
